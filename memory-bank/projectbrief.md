# Project Brief: Academy

## Overview
Self-hosted coding assessment platform for technical interviews. Replace expensive SaaS tools ($5,000-15,000/year) with a simple, one-time setup solution.

## Core Requirements
- Handle 300 concurrent candidates taking exams simultaneously
- Support Python, Java, C++, Go code execution
- Real-time code execution with Judge0 CE
- Medium proctoring (tab switch, copy/paste tracking)
- Admin dashboard for problem/exam management
- Results export with candidate scores and code submissions

## Success Metrics
- 300 candidates can start exam simultaneously
- Code execution completes within 15 seconds
- Zero data loss during exam
- Results exportable within 1 minute of exam end

## Constraints
- Single-person setup and maintenance
- One-time build, minimal ongoing work
- Must be self-hosted (data ownership)
- Budget: ~$50-100/month for hosting
