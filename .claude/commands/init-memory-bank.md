# Initialize Memory Bank

Set up the memory bank for a new project or reset it.

## Instructions

Create or reset the memory-bank/ directory with these files:

1. **projectbrief.md** - Ask user about:
   - Project name and purpose
   - Core requirements
   - Success metrics
   - Constraints

2. **techContext.md** - Document:
   - Tech stack with versions
   - Key integrations
   - Development commands
   - Environment variables

3. **systemPatterns.md** - Document:
   - Architecture patterns
   - Code conventions
   - Common patterns used

4. **activeContext.md** - Initialize:
   - Current phase
   - What we're working on
   - Recent decisions
   - Next steps

5. **progress.md** - Initialize:
   - Completed milestones (empty)
   - Current sprint tasks
   - Known issues (empty)

## Output
Confirm memory bank is initialized and ready.
