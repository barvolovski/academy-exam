# Plan Implementation

Create a detailed plan for the requested task.

## Instructions

Based on the current context (run /workflow-understand first if needed):

1. **Analyze the request**
   - What exactly needs to be built?
   - What are the acceptance criteria?

2. **Check existing patterns**
   - Review memory-bank/systemPatterns.md
   - Follow established conventions

3. **Break down into steps**
   - List specific files to create/modify
   - Order by dependencies
   - Identify potential blockers

4. **Update activeContext.md**
   - Add the plan to "What We're Working On"
   - Note any decisions made

## Output Format

```markdown
## Plan: [Task Name]

### Files to Create/Modify
1. path/to/file.ts - Description
2. ...

### Implementation Steps
1. Step description
2. ...

### Potential Issues
- Issue and mitigation
```
